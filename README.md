# AI Financial Adviser

A machine learning system that analyzes personal financial data and provides personalized financial recommendations.

## Features

- Financial risk profiling using ML
- Financial health score
- Personalized investment advice
- Interactive Streamlit dashboard

## Tech Stack

Python  
Pandas  
Scikit-learn  
Streamlit  
Machine Learning

## Project Workflow

Data Collection → Data Processing → Feature Engineering → Model Training → Recommendation Engine → Web App

## Machine Learning Model

Random Forest Classifier used to predict financial risk tolerance.

## How to Run

1. Train the model

python src/model_training.py

2. Run the dashboard

streamlit run app/streamlit_app.py
